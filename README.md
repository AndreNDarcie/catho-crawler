# Catho Crawler

NodeJS Crawler for [Catho](http://www.catho.com.br/vagas/por-area/).     
- Available in: [Heroku](https://catho-crawler.herokuapp.com/)

## Setup

You can clone from github:

    git clone git@github.com:andrendarcie/catho-crawler.git

    cd catho-crawler

    npm install
